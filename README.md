# JEAF X-Fun Types README #

This repository contains a collection of basic types that are used quite often in enterprise applications.
 

## Links ##
For further information please refer to:

* [JEAF X-Fun Getting started](https://anaptecs.atlassian.net/l/c/8vuqYGsN)
* [Maven Repository](https://search.maven.org/artifact/com.anaptecs.jeaf.x-fun/jeaf-x-fun-types)
* [Javadoc](https://javadoc.io/doc/com.anaptecs.jeaf.x-fun/jeaf-x-fun-types)

## How do I get set up? ##

* Create a clone of this repository on your local machine.
* Execute Maven on the top level project `maven clean install`
